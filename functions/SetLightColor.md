# SetLightColor

Sets the current color of the light, as an RGB vector.

Example:
```c++
self SetLightColor( (0.5, 0.75, 1.0) )
```

### Parameters
Parameter | Summary
| - | - |
*color | An RGB vector with components in the range 0 to 1. The given values get rounded before they are stored. (vector3)

<sub>* = mandatory</sub>
