# EnableFrontendStreamingOverlay

Enable or disable the draw of an alternate draw mode for high detail entities marked to use alt draw

Example:
```c++
entity EnableFrontendStreamingOverlay( 0, 0 )
```

### Parameters
Parameter | Summary
| - | - |
*localClientNum | 
*enable | 0-disable 1-enable
extracam | 0-disable 1-enable - when enabled the overlay is drawn on extracam

<sub>* = mandatory</sub>
