# MoveX

Move this entity to the given world x value

Example:
```c++
train MoveX( -4400, 60, 15, 20 )
```


### Parameters
Parameter | Summary
| - | - |
*point | The x value to move the entity to, as a floating point number
*time | The time to move the entity in seconds
[acceleration time] | The time spent accelerating in seconds
[deceleration time] | The time spent decelerating in seconds

<sub>* = mandatory</sub>
