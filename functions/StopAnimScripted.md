# StopAnimScripted

Halts animscript on this entity.

Example:
```c++
self StopAnimScripted()
```


### Parameters
Parameter | Summary
| - | - |
[blend] | The blend out time for the animation
[ImmediateStop] | Stop the animation immediatly on the client side (no client/server time sync)

<sub>* = mandatory</sub>
