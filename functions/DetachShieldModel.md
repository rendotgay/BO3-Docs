# DetachShieldModel

Detaches an attached shield model from a player.

Example:
```c++
self DetachShieldModel( "weapon_riot_shield", "tag_weapon_left" )
```


### Parameters
Parameter | Summary
| - | - |
*modelname | The model name to detach (string)
[tagname] | The tag to detach the model from (string)

<sub>* = mandatory</sub>
