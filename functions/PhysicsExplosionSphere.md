# PhysicsExplosionSphere

Create a physics explosion sphere

Example:
```c++
PhysicsExplosionSphere( origin, radius, radius, 5, max_damage, min_damage )
```

### Parameters
Parameter | Summary
| - | - |
*position | The origin of the sphere
*outer radius | The outer radius of the sphere
*inner radius | The inner radius of the sphere
*magnitude | The magnitude of the explosion
[outer damage] | The optional outer damage
[inner damage] | The optional inner damage

<sub>* = mandatory</sub>
