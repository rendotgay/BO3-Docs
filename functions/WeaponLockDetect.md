# WeaponLockDetect

Sets the tartget as detected.  Will clear any existing hard lock.  Use WeaponLockStart() to start lock.

Example:
```c++
player WeaponLockDetect( enemyGuy )
```

### Parameters
Parameter | Summary
| - | - |
*entity | : An entity.
[slot] | :  The slot to operate on (up to MAX_SIMULTANEOUS_LOCKONS)

<sub>* = mandatory</sub>
