# IsFriendly

Returns true if entity is friendly

Example:
```c++
if ( entity IsFriendly( &lt;localClientNum&gt;, [predicted] ) )
```


### Parameters
Parameter | Summary
| - | - |
*localclientnum | Local client number of the local client you want to test against
[predicted] | Test against predicted version, false by default

<sub>* = mandatory</sub>
