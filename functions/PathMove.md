# PathMove

Moves the entire path for this vehicle to the specific origin and angles.

Example:
```c++
huey PathMove( node, origin, angles )
```


### Parameters
Parameter | Summary
| - | - |
*node index | Which node to move to the new location (it moves the whole path not just this node).
*new origin | The new position
*new angles | The new angles

<sub>* = mandatory</sub>
