# SetClockUp

Set a clock HUD element to count up over a time period.

Example:
```c++
level.bombstopwatch SetClockUp( 5, 60, "hudStopwatch", 64, 64 )
```

### Parameters
Parameter | Summary
| - | - |
*time | The new timer time in seconds to count up
*full time | The time in seconds represented by a full cycle of the hand, for instance a stopwatch would be 60s.
*material | The material for the hud clock
[width] | The width of the material.
[height] | The height of the material.

<sub>* = mandatory</sub>
