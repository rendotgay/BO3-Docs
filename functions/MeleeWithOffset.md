# MeleeWithOffset

Makes this actor melee attack. Returns the entity hit, if any.

Example:
```c++
guy MeleeWithOffset( (0,0,32) )
```

**Returns:** [`entity`](../types/entity.md)

### Parameters
Parameter | Summary
| - | - |
*offset | A vector offset added to the actor's eye position from which to start the melee trace.

<sub>* = mandatory</sub>
