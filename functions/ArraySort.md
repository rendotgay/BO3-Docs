# ArraySort

Returns a sorted array of entities by closest or farthest distance from &lt;origin&gt

Example:
```c++

```

### Parameters
Parameter | Summary
| - | - |
*array | array to sort
*origin | the origin from which to compare
[closest] | if set to true the array will be sorted by distance closest to <origin>, if false by distance farthest from <origin>
[max] | if set then only the first <max> count of entities will be returned
[range] | if set then only the entities within <range> will be considered

<sub>* = mandatory</sub>
