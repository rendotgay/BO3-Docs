# GetActorSpawnerTeamArray

Returns an array of all of the spawners in a level

**Returns:** [`entity`](../types/entity.md)

### Parameters
Parameter | Summary
| - | - |
*team | a team name, either 'axis', 'allies', or 'neutral'
[team] | any number of additional team names may be added, either 'axis', 'allies', or 'neutral'

<sub>* = mandatory</sub>
