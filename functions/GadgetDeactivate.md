# GadgetDeactivate

turns gadget off in slot.

Example:
```c++
self GadgetDeactivate( slot, weapon, GADGET_OFF_PENALTY_SHUT_OFF )
```

### Parameters
Parameter | Summary
| - | - |
*slot | The gadget slot to deactivate. (integer)
*weapon | The gadget weapon
penalty | The power penalty type

<sub>* = mandatory</sub>
