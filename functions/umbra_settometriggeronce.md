# umbra_settometriggeronce

Set the active tome trigger without interferring with any other tome logic.

Example:
```c++
Umbra_SetTomeTriggerOnce( localClientNum, "my_really_cool_tome_trigger" )
```

### Parameters
Parameter | Summary
| - | - |
*localClientNum | 
*tometrigger name | name of a tome trigger. Use empty string "" to deactivate tome triggers

<sub>* = mandatory</sub>
