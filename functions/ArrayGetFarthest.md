# ArrayGetFarthest

Returns the entity/vector that is farthest to the &lt;origin&gt; from the &lt;array&gt;.

Example:
```c++
ent = ArrayGetFarthest( player.origin, a_enemies )
```

### Parameters
Parameter | Summary
| - | - |
*origin | The origin to get farthest from
*array | The array to search in
[dist] | The maximum distance when finding farthest (has to be positive)

<sub>* = mandatory</sub>
