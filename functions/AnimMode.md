# AnimMode

Set the way that animation deltas are interpreted by the game engine

Example:
```c++
self AnimMode( "gravity" )
```

### Parameters
Parameter | Summary
| - | - |
*mode | which animmode to use. Must be 'gravity', 'nogravity', 'angle deltas', 'pos deltas', 'normal', 'normal_nogravity', 'zonly_physics', 'nophysics', 'none'
[clear path] | The clear path flag

<sub>* = mandatory</sub>
