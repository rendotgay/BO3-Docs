# OED_SitRepScan_SetOutline

Set the outline level for the sitrep scan

Example:
```c++
entity oed_sitrepscan_setoutline( outlinelevel )
```


### Parameters
Parameter | Summary
| - | - |
*outlinelevel | The outline level (float)
AlphaType | 0-default 1-Alpha ( 0 by default )

<sub>* = mandatory</sub>
