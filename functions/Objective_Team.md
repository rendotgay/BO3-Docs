# Objective_Team

Sets the team that the objective is for. Allows having different objectives for each team

Example:
```c++
Objective_Team( 0, "allies" )
```


### Parameters
Parameter | Summary
| - | - |
*objective_number | The ID of the objective to alter
*team | The team that the objective is for. Valid entries are 'allies', 'axis', 'team3', or 'none'

<sub>* = mandatory</sub>
