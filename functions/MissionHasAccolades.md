# MissionHasAccolades

Returns whether or not this mission has any accolades associated with it

Example:
```c++
level.has_accolades = MissionHasAccolades( "cp_mi_cairo_ramses2" )
```


### Parameters
Parameter | Summary
| - | - |
[currentmap] | the name of the current map.  If ommitted, use Dvar sv_mapname

<sub>* = mandatory</sub>
