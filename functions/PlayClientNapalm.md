# PlayClientNapalm

Plays the client sided part of the Napalm

Example:
```c++
PlayClientNapalm( startPos, plane.yaw, "vietcong", "axis", plane.owner, "left", height )
```

### Parameters
Parameter | Summary
| - | - |
*position | The position of Napalm
*yaw | The yaw
*teamFaction | The team faction
*team | The team
*owner | The owner of Napalm
*exittype | The exit type
*height | The height

<sub>* = mandatory</sub>
