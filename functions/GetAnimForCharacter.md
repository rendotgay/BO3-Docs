# GetAnimForCharacter

Returns the name of the animation to use for the character pose in the frontend

Example:
```c++
frontendVignetteAnim = GetAnimForCharacter( characterIndex, "mp" )
```

### Parameters
Parameter | Summary
| - | - |
*characterIndex | The index of the character in the current game-mode's character table
*sessionmode | Session Mode Index

<sub>* = mandatory</sub>
