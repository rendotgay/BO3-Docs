# SetHintString

Set the hint string for the entity

Example:
```c++
t_door SetHintString( "Press and Hold ^3[{+activate}]^7 to transmit unlock code" )
```

### Parameters
Parameter | Summary
| - | - |
*hint string | The hint string that will be displayed to the player
[...] Additional arguments may be appended to the hint string | 

<sub>* = mandatory</sub>
