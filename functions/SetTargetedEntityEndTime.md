# SetTargetedEntityEndTime

Set targeted entity end time, for use on the client side in the UI.

Example:
```c++
self SetTargetedEntityEndTime( weapon, 0.0 )
```


### Parameters
Parameter | Summary
| - | - |
*weapon | The weapon (player should have this weapon)
*endtime | The end time of the targeted entity

<sub>* = mandatory</sub>
