# SetCameraSpikeCamAngles

Sets the orientation of a client's extra cam.

Example:
```c++
SetCameraSpikeCamAngles( localClientNum, angles )
```

### Parameters
Parameter | Summary
| - | - |
*localClientNum | Client number of the camera-spike cam owner
*angles | New pitch, yaw, and roll of the camera-spike cam

<sub>* = mandatory</sub>
