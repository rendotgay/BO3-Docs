# SubtitlePrint

print to the subtitle channel

Example:
```c++
SubtitlePrint( 0, 100, "zombies everywhere")
```


### Parameters
Parameter | Summary
| - | - |
*localClientNum | The localClientNum to flush the subtitles for
*msec | The time to display the subtitle for
*subtitle | The message to display

<sub>* = mandatory</sub>
