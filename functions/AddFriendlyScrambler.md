# AddFriendlyScrambler

Adds new friendly scrambler to the compass

Example:
```c++
player AddFriendlyScrambler( scrambler.origin[0], scrambler.origin[1], scrambler.handle )
```

### Parameters
Parameter | Summary
| - | - |
*x | X coord (float)
*y | Y coord (float)
*handle | will be used to remove the scrambler (integer)

<sub>* = mandatory</sub>
