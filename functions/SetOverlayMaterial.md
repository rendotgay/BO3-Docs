# SetOverlayMaterial

Set the overlay material

Example:
```c++
SetOverlayMaterial( self.localClientNum, 2, 20, 1 )
```


### Parameters
Parameter | Summary
| - | - |
*localClientNum | 
*overlayIndex | Must be between 0 - 3
*materialIndex | Must be between 0 - 63
*sortIndex | Must be between 0 - 2

<sub>* = mandatory</sub>
