# StrTok

Tokenizes &lt;string&gt; by the delimiters &lt;delim&gt;. Returns the array of string tokens.

Example:
```c++
my_string_array = StrTok( "my_string; my_string_2", "; " )
```


### Parameters
Parameter | Summary
| - | - |
*string | The input string
*delim | The delimiter to tokenize by

<sub>* = mandatory</sub>
