# AddInfluencer

Adds a new influencer of &lt;name&gt; at &lt;origin&gt

Example:
```c++
influencer_id = AddInfluencer( name, origin, allies_mask )
```


### Parameters
Parameter | Summary
| - | - |
*name | valid spawn influencer preset name.
*origin | center point of the influencer
*team mask | teams that this influencer will affect

<sub>* = mandatory</sub>
