# GetTopPlayersTeam

Returns the team of the player top player in the game by index

Example:
```c++
team = GetTopPlayersTeam( localClientNum, 2 )
```


### Parameters
Parameter | Summary
| - | - |
*localClientNum | The localClientNum of the player in question
*topPlayerIndex | The index of the top player

<sub>* = mandatory</sub>
