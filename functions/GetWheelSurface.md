# GetWheelSurface

Returns the surface type of the given wheel as a string.

Example:
```c++
surface = self GetWheelSurface( side )
```

### Parameters
Parameter | Summary
| - | - |
*wheel | The wheel position, can be 'front_left', 'front_right', 'back_left', 'back_right', 'middle_left', 'middle_right'

<sub>* = mandatory</sub>
