# SetFilterPassQuads

Sets the filter pass quads

Example:
```c++
SetFilterPassQuads( self.localClientNum, 1, 12, 200)
```


### Parameters
Parameter | Summary
| - | - |
*localClientNum | 
*filterIndex | Must be between 0 - 3
*passIndex | Must be between 0 - 16
*passQuads | Must be between 0 - 2048

<sub>* = mandatory</sub>
