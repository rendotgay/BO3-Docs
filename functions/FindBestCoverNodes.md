# FindBestCoverNodes

Find the best cover nodes for a given NPC given his state. Sorted by best

Example:
```c++
nodes = self FindBestCoverNodes(radius,origin)
```


**Returns:** [`pathnode`](../types/pathnode.md)

### Parameters
Parameter | Summary
| - | - |
*radius | search radius
*origin | search origin

<sub>* = mandatory</sub>
