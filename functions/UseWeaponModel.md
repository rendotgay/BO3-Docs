# UseWeaponModel

Use a weapon model

Example:
```c++
wall_weapon_model UseWeaponModel("m16_zm")
```


### Parameters
Parameter | Summary
| - | - |
*weaponName | The name of the weapon to base the tags to hide on
[model] | The name of a model to use. If none is specified the world model for the weapon will be used
[weapon options] | The weapon options

<sub>* = mandatory</sub>
