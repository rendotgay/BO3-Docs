# NewTeamHudElem

Create a new hud element for a particular team

Example:
```c++
level.progressbar_allies_neutralize = NewTeamHudElem( "axis" )
```


**Returns:** [`hudelem`](../types/hudelem.md)

### Parameters
Parameter | Summary
| - | - |
*team | The team for whom the hud element is created. Must be 'axis', 'allies' or 'spectator'

<sub>* = mandatory</sub>
