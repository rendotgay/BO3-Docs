# LinkToMissile

Links a player camera and controls to a missile

Example:
```c++
player LinkToMissile( rocket, true )
```

### Parameters
Parameter | Summary
| - | - |
*missile | entity to link the player to
[boost_enabled] | : If true, the player can trigger missile boost
[brake_enabled] | : If true, the player can trigger missile brake

<sub>* = mandatory</sub>
