# StrEndsWith

Returns true/false if &lt;string&gt; ends with &lt;substring&gt;. Case sensitive.

Example:
```c++
ends_with = StrEndsWith( "my_string", "_string" )
```


### Parameters
Parameter | Summary
| - | - |
*string | String to search
*substring | Substring to search with.

<sub>* = mandatory</sub>
