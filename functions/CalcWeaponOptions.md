# CalcWeaponOptions

Calculates weapon options

Example:
```c++
primaryWeaponOptions = self CalcWeaponOptions( class_num, 0 )
```


### Parameters
Parameter | Summary
| - | - |
*custom class slot | The custom class slot
*weapon slot | primary weapon=0 or secondary weapon=1 or knife weapon=2

<sub>* = mandatory</sub>
