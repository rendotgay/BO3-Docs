# VehDriverAimAtCrosshairs

Enables or disables the driver turret aiming at the crosshairs (aiming at the crosshairs overrides all other aim targets and is the default behavior)

Example:
```c++
vehicle VehDriverAimAtCrosshairs( false )
```

### Parameters
Parameter | Summary
| - | - |
*AimAtCrosshairs | True to aim the turret at the crosshairs

<sub>* = mandatory</sub>
