# GetTopPlayersWeaponInfo

Returns the equipped weapon of the top player in the game

Example:
```c++
weapon = GetTopPlayersWeaponInfo( localClientNum, 2 )
```

### Parameters
Parameter | Summary
| - | - |
*localClientNum | The localClientNum of the player in question
*topPlayerIndex | The index of the top player

<sub>* = mandatory</sub>
