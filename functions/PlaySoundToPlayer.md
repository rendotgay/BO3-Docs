# PlaySoundToPlayer

Play the sound alias as if coming from the entity, so that only one player can hear it

Example:
```c++
self PlaySoundToPlayer( "frag_out", self )
```

### Parameters
Parameter | Summary
| - | - |
*aliasname | The sound alias to play (string) or the sound alias (int)
*player | The player that will be able to hear the sound. (entity)

<sub>* = mandatory</sub>
