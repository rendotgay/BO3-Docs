# CarryTurret

Links the auto turret to the player to carry around.

Example:
```c++
player CarryTurret( turret, offset, angleOffset )
```


### Parameters
Parameter | Summary
| - | - |
*turret | The turret entity
*offset | the offset from the player to place the turret
*angleOffset | the angle offset of the turret

<sub>* = mandatory</sub>
