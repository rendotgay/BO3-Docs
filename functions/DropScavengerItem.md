# DropScavengerItem

Drop an item with the given item name. This item will be only available to those with the scavenger perk

Example:
```c++
self DropScavengerItem( "scavenger_item_mp" )
```

**Returns:** [`entity`](../types/entity.md)

### Parameters
Parameter | Summary
| - | - |
*item name | the name of the item to drop

<sub>* = mandatory</sub>
