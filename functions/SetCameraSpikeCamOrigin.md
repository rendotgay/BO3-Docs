# SetCameraSpikeCamOrigin

Sets the origin of a client's camera-spike cam

Example:
```c++
SetCameraSpikeCamOrigin( localClientNum, origin )
```


### Parameters
Parameter | Summary
| - | - |
*localClientNum | Client number of the camera-spike cam owner
*origin | New origin of the camera-spike cam

<sub>* = mandatory</sub>
