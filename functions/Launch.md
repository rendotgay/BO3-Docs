# Launch

Launch an object that interacts with the world, using an initial velocity. From this point on this object will no longer block either missiles or bullets.

Example:
```c++
self Launch( (x, y, z) )
```


### Parameters
Parameter | Summary
| - | - |
*initial velocity | The initial velocity of the launch.
[initial angular velocity] | The initial angular velocity of the launch.

<sub>* = mandatory</sub>
