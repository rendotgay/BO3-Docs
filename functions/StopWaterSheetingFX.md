# StopWaterSheetingFX

Stop the fullscreen water effect

Example:
```c++
StopWaterSheetingFX( localClientNum, 1 )
```


### Parameters
Parameter | Summary
| - | - |
*localClientNum | Local client on which to play the fx
[duration] | The number of seconds it takes the effect to fade. Use 0 for immediate turn off. Defaults to 0.

<sub>* = mandatory</sub>
