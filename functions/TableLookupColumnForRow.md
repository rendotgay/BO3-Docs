# TableLookupColumnForRow

look up a particular row and column in a table

Example:
```c++
TableLookupColumnForRow( "gamedata/tables/mp/exampleStringTable.csv", 101, 7 )
```

### Parameters
Parameter | Summary
| - | - |
*filename | The table to look up
*search row num | The row number of the string table to search through
*search column num | The column number of the string table to search through

<sub>* = mandatory</sub>
