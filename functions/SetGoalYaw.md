# SetGoalYaw

Set the goal yaw direction for this vehicle. Goal yaw is ignored if vehicle doesn't stop at goal. Lookat entity has priority over goal yaw

Example:
```c++
level.flak1 SetGoalYaw( int_value )
```

### Parameters
Parameter | Summary
| - | - |
*integer | The yaw direction of the target: 0 to 360 integer value

<sub>* = mandatory</sub>
