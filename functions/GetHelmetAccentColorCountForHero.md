# GetHelmetAccentColorCountForHero

Returns the number of accents for the specified helmet model of a specific hero

Example:
```c++
accentColorCount = GetHelmetAccentColorCountForHero( localClientNum, 0, 0 )
```


### Parameters
Parameter | Summary
| - | - |
*localClientNum | The localClientNum of the player in question
*heroIndex | The index of the hero in the current game-mode's character table
*helmetIndex | The index of the helmet for the specific hero

<sub>* = mandatory</sub>
