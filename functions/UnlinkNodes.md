# UnlinkNodes

Disconnects node1 from node2. Call it again with arguments flipped if you want to completely disconnect the two nodes.

Example:
```c++
UnlinkNodes( node_moving_elev_left, node_moving_elev_right )
```


### Parameters
Parameter | Summary
| - | - |
*node1 | first node to unlink
*node2 | second node to unlink

<sub>* = mandatory</sub>
