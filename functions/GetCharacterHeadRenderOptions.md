# GetCharacterHeadRenderOptions

Calculates the packed helmet render option - to be passed to SetBodyRenderOptions()

Example:
```c++
helmetRenderOptions = GetCharacterHeadRenderOptions( 0 )
```


### Parameters
Parameter | Summary
| - | - |
*headIndex | The index of the character in the current game-mode's character table

<sub>* = mandatory</sub>
