# SetCameraSpikeCamActive

Sets whether a client's camera-spike cam is active.

Example:
```c++
SetCameraSpikeCamActive( localClientNum, 1 )
```

### Parameters
Parameter | Summary
| - | - |
*localClientNum | Client number of the extra cam owner
*active | 0 if inactive, 1 if active

<sub>* = mandatory</sub>
