# LerpFloat

Returns a lerped value between &lt;from&gt; and &lt;to&gt; based on &lt;time&gt

Example:
```c++
val = LerpFloat( 0, 10, 0.5);	// would return 5.0
```


### Parameters
Parameter | Summary
| - | - |
*from | Floating point value, start of the range
*to | Floating point value, end of the range
*time | Floating point value in the range of 0 - 1

<sub>* = mandatory</sub>
