# IsWheelPeelingOut

Returns if the given wheel is sliding.

Example:
```c++
peeling = self IsWheelPeelingOut( front_left )
```

### Parameters
Parameter | Summary
| - | - |
*wheel | The wheel position, can be 'front_left', 'front_right', 'back_left', 'back_right', 'middle_left', 'middle_right'

<sub>* = mandatory</sub>
