# StrStartsWith

Returns true/false if &lt;string&gt; starts with &lt;substring&gt;. Case sensitive.

Example:
```c++
starts_with = StrStartsWith( "my_string", "my_" )
```


### Parameters
Parameter | Summary
| - | - |
*string | String to search
*substring | Substring to search with.

<sub>* = mandatory</sub>
