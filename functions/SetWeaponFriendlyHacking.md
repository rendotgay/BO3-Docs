# SetWeaponFriendlyHacking

Set the weapon lockon completion percentage

Example:
```c++
self SetWeaponFriendlyTargetLocked( weapon, 0.0 )
```


### Parameters
Parameter | Summary
| - | - |
*weapon | The weapon (player should have this weapon)
*isFriendlyHacking | Set to true if a friendly is hacking the target

<sub>* = mandatory</sub>
