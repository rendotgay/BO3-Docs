# SpawnVehicle

Spawns a new vehicle and returns a reference to it



**Returns:** [`entity`](../types/entity.md)

### Parameters
Parameter | Summary
| - | - |
*vehicletype | vehicle type (constant string)
*origin | The position of the vehicle (vector)
*angles | The angle to spawn the vehicle at (vector)
[targetname] | vehicle target name (constant string)
[destructibledef] | The destructibledef to use to make this vehicle a destructible

<sub>* = mandatory</sub>
