# DoesWeaponReplaceSpawnWeapon

Returns true if the new weapon is an alt mode of the spawn weapon. Only certain alt variations are valid (not-gl)

Example:
```c++
if ( DoesWeaponReplaceSpawnWeapon( self.spawnWeapon, newWeapon ) )
```

### Parameters
Parameter | Summary
| - | - |
*weapon name | 

<sub>* = mandatory</sub>
