# GetCyberComWeapon

Gets the cybercom gadget weapon given the flag, type, and if upgraded

Example:
```c++
GetCyberComWeapon(  flag, type, upgraded )
```

### Parameters
Parameter | Summary
| - | - |
*flag | flag field
*type | type of weapon
*upgraded | true if the weapon is upgraded

<sub>* = mandatory</sub>
