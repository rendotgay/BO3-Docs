# SetCameraSpikeCamEntity

Sets an camera-spike cam entity for the client.

Example:
```c++
CScr_SetCameraSpikeCamEntity( localClientNum, cameraSpike )
```

### Parameters
Parameter | Summary
| - | - |
*localClientNum | Client number of the extra cam owner
*entity | Camera entity, or undefined

<sub>* = mandatory</sub>
