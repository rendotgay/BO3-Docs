# GetNodeArray

Gets an array of nodes that have the given name, key pair

Example:
```c++
node = GetNodeArray( self.target, "targetname" )
```

**Returns:** [`pathnode`](../types/pathnode.md)

### Parameters
Parameter | Summary
| - | - |
*name | name to search for
*key | key that name goes with

<sub>* = mandatory</sub>
