# SetCorpseGibState

Sets the gib state for the body rendering.

Example:
```c++
corpse SetCorpseGibState( leg_state, arm_state )
```

### Parameters
Parameter | Summary
| - | - |
*leg state | 0 - 3 state: 0 is no gib, 1 right, 2 left, 3 both
*arm state | 0 - 3 state: 0 is no gib, 1 right, 2 left, 3 both

<sub>* = mandatory</sub>
