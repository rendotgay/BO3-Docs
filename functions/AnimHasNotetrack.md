# AnimHasNotetrack

queries the given animation for a note track

Example:
```c++
if ( IsDefined( facialanim ) && AnimHasNotetrack( facialanim, "dialogue" ) ) ...
```

### Parameters
Parameter | Summary
| - | - |
*animation | an animation
*note track | a constant string with the name of the note track

<sub>* = mandatory</sub>
