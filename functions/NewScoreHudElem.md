# NewScoreHudElem

Create a new score hud element for a particular client

Example:
```c++
self.hud_rankscroreupdate = NewScoreHudElem(self)
```


**Returns:** [`hudelem`](../types/hudelem.md)

### Parameters
Parameter | Summary
| - | - |
*client | The client for whom the hud element is created.

<sub>* = mandatory</sub>
