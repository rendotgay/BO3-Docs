# StopRevive

Stops the revive behavior on a player when the revive did not complete. &lt;reviver&gt; is the one administering aid to the player this is called on.

Example:
```c++
playerInLastStand StopRevive( rescuer )
```


### Parameters
Parameter | Summary
| - | - |
*reviver | the player reviving the player this is called on

<sub>* = mandatory</sub>
