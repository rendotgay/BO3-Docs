# SetOverlayConstant

Set the overlay constant

Example:
```c++
SetOverlayConstant( self.lcoalClientNum, 2, 5, 1.4 )
```


### Parameters
Parameter | Summary
| - | - |
*localClientNum | 
*overlayIndex | Must be between 0 - 3
*constantIndex | Must be between 0 - 16
*value | The value to set

<sub>* = mandatory</sub>
