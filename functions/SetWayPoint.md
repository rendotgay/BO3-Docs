# SetWayPoint

Sets a hud element to be a waypoint.

Example:
```c++
newdeathicon SetWayPoint( true )
```


### Parameters
Parameter | Summary
| - | - |
*constantSize | boolean describing whether the hud element remains the same size
[offscreenMaterialName] | Material to draw when waypoint is offscreen.
[isDistanceShown] | boolean turning on/off player distance to waypoint display above the waypoint.
[no offscreen] | boolean turning on/off arrow indicators when element is offscreen.

<sub>* = mandatory</sub>
