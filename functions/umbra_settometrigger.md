# umbra_settometrigger

Sets current active tome trigger

Example:
```c++
Umbra_SetTomeTrigger( localClientNum, "my_really_cool_tome_trigger" )
```


### Parameters
Parameter | Summary
| - | - |
*localClientNum | 
*tometrigger name | name of a tome trigger. Use empty string "" to deactivate tome triggers

<sub>* = mandatory</sub>
