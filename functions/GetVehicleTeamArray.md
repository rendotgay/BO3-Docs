# GetVehicleTeamArray

Returns an array of all of the vehicles in a level that are not sentients

Example:
```c++
enemies = GetVehicleTeamArray( "axis", "neutral" )
```

### Parameters
Parameter | Summary
| - | - |
[team] | any number of teams names may be added, either 'axis', 'allies', or 'neutral'

<sub>* = mandatory</sub>
