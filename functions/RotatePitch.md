# RotatePitch

Rotate this entity to the given pitch

Example:
```c++
treeorg RotatePitch( -5, 0.26, 0.15, 0.1 )
```

### Parameters
Parameter | Summary
| - | - |
*pitch angle | The new pitch angle in degrees
*time | The time to rotate in seconds
[acceleration time] | The time spent accelerating in seconds
[deceleration time] | The time spent decelerating in seconds

<sub>* = mandatory</sub>
