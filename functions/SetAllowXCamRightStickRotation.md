# SetAllowXCamRightStickRotation

Prevent or stop preventing right stick rotation on this xcam (the next playmaincamxcam resets this)

Example:
```c++
SetAllowXCamRightStickRotation( 0, false )
```


### Parameters
Parameter | Summary
| - | - |
*localClientNum | The local client
*newValue | True to allow, False, to disable right stick rotation

<sub>* = mandatory</sub>
