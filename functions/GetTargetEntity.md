# GetTargetEntity

Gets the target entity of this turret or vehicle weapon

Example:
```c++
target = roof_turret GetTargetEntity( ); target = tank GetTargetEntity( )
```


**Returns:** [`entity`](../types/entity.md)

### Parameters
Parameter | Summary
| - | - |
[gunnerIndex] | the gunner index if called on a vehicle

<sub>* = mandatory</sub>
