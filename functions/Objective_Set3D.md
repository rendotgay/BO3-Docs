# Objective_Set3D

Sets flag determining if 3D state for an objective is drawn.

Example:
```c++
Objective_Set3D( myObjNum, true, undefined, undefined, undefined, (0,0,70) )
```


### Parameters
Parameter | Summary
| - | - |
*objective_number | The ID of the objective to alter
*use3D | true if should render in hud
[unused] | not used
[unused] | not used
[unused] | not used
[unused] | not used

<sub>* = mandatory</sub>
