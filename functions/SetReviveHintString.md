# SetReviveHintString

Set the revive hint string to show to all or to a team.

Example:
```c++
self.revivetrigger setReviveHintString( &"GAME_BUTTON_TO_REVIVE_PLAYER", self.team )
```


### Parameters
Parameter | Summary
| - | - |
*hint string | The hint string that will be displayed to the player that has the perk
[team] | The team to show the hint to

<sub>* = mandatory</sub>
