# DiffTrackAngle

Returns a lagged angle between &lt;from&gt; and &lt;to&gt; based on &lt;rate&gt; and &lt;deltaTime&gt;. Properly accounts for wrapping 2pi

Example:
```c++

```

### Parameters
Parameter | Summary
| - | - |
*to | Float, start of the range
*from | Float, end of the range
*rate | Floating point
*deltaTime | Time elapsed

<sub>* = mandatory</sub>
