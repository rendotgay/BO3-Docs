# ItemWeaponSetAmmo

Set the weapon ammo to the given clip ammo and reserve ammo. Can give an alternate weapon index

Example:
```c++
weap ItemWeaponSetAmmo( clip, extra, 1 )
```


### Parameters
Parameter | Summary
| - | - |
*clipAmmo | Ammo for the clip
*reserveAmmo | Reserve ammo
[altIndex] | Alternate weapon index

<sub>* = mandatory</sub>
