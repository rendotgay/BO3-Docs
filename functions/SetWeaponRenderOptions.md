# SetWeaponRenderOptions

Sets the weapon render options in the entity.

Example:
```c++
level.weapon_clientscript_model SetWeaponRenderOptions( level.camo_index, level.reticle_index, level.show_player_tag, level.show_emblem, level.show_paintshop )
```
