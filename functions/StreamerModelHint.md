# StreamerModelHint

Force a model to be loaded for a set period of time.

Example:
```c++
streamerModelHint( "c_hro_hendricks_base_fb", 10 )
```

### Parameters
Parameter | Summary
| - | - |
*modelname | The name of the model.
*duration | Length of time (in seconds) to keep it loaded.

<sub>* = mandatory</sub>
