# GetWeaponAccuracy

Returns accuracy at the distance from the current enemy.

Example:
```c++
getweaponaccuracy( self, "ak47" ), getweaponaccuracy( ai, ai.primaryweapon )
```


### Parameters
Parameter | Summary
| - | - |
*entity | An entity. Must be AI.
*weapon name | The weapon name.

<sub>* = mandatory</sub>
