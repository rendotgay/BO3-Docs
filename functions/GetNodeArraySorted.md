# GetNodeArraySorted

Gets an array of nodes that have the given name, key pair, within given radius, sorts them

Example:
```c++
nodes = GetNodeArraySorted(self.target, "targetname", self.origin, 256 )
```

**Returns:** [`pathnode`](../types/pathnode.md)

### Parameters
Parameter | Summary
| - | - |
*name | name to search for
*key | key that name goes with
*pos | search origin
*r | the radius to sort in

<sub>* = mandatory</sub>
