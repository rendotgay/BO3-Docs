# AddZombieBoxWeapon

Adds a zombie box weapon

Example:
```c++
AddZombieBoxWeapon( weapon, weapon.worldModel, weapon.isDualWield )
```

### Parameters
Parameter | Summary
| - | - |
*weapon | The weapon to add
*weaponModelName | The model name of the weapon
*isDualWield | True if the weapon is dual wield, false otherwise

<sub>* = mandatory</sub>
