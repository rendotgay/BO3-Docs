# SetFilterBitFlag

Set or clear the filter bit flag

Example:
```c++
SetFilterBitFlag( self.localClientNum, 1, 10 )
```

### Parameters
Parameter | Summary
| - | - |
*localClientNum | 
*filterIndex | Must be between 0 - 3
*bitIndex | Must be between 0 - 31
*setOrClear | Must be between 0 - 1

<sub>* = mandatory</sub>
