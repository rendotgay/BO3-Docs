# GetTopPlayersGesture

Returns the selected gesture of type CharacterGestureTypes for the given player

Example:
```c++
gesture_anim = GetTopPlayersGesture( localClientNum, 2, 2 );	// GESTURE_TYPE_BOAST = 2
```

### Parameters
Parameter | Summary
| - | - |
*localClientNum | The localClientNum of the player in question
*topPlayerIndex | The index of the top player
*CharacterGestureTypes | The CharacterGestureTypes enum value of the gesture to use

<sub>* = mandatory</sub>
