# WeaponHasAttachmentAndUnlocked

Does the weapon have all the attachments specified? And are they all unlocked for challenge stat purposes?

Example:
```c++
player WeaponHasAttachmentAndUnlocked( weapon, "extbarrel", "damage, "fmj" )
```


### Parameters
Parameter | Summary
| - | - |
*weapon | weapon
*string | string id for attachment

<sub>* = mandatory</sub>
