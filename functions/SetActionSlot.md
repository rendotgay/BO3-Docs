# SetActionSlot

Sets action slot for the player

Example:
```c++
self SetActionSlot( 4, "weapon", level.DEAD_CONTROL_WEAPON )
```


### Parameters
Parameter | Summary
| - | - |
*slot | The action slot
*action slot type | The type of slot. can be "weapon"/"altMode"/""
[weapon] | The weapon to set

<sub>* = mandatory</sub>
