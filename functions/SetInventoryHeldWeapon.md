# SetInventoryHeldWeapon

Sets the player's inventoryHeldWeapon.

Example:
```c++
player SetInventoryHeldWeapon( weapon )
```


### Parameters
Parameter | Summary
| - | - |
*weapon | The held weapon to be set as inventoryHeldWeapon. Use undefined to clear the inventoryHeldWeapon.

<sub>* = mandatory</sub>
