# PhysicsLaunch

Permanently turn this entity into a physics object with an intial force vector at the specified point.

Example:
```c++
dummy PhysicsLaunch( contact_point, initial_force )
```


### Parameters
Parameter | Summary
| - | - |
[contact_point] | The point to apply the initial force
[initial_force] | The force vector to apply

<sub>* = mandatory</sub>
