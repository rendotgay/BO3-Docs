# SetFilterPassMaterial

Sets the filter pass material

Example:
```c++
SetFilterPassMaterial( self.localClientNum, 1, 12, 20 )
```


### Parameters
Parameter | Summary
| - | - |
*localClientNum | 
*filterIndex | Must be between 0 - 3
*passIndex | Must be between 0 - 15
*materialIndex | Must be between 0 - 63

<sub>* = mandatory</sub>
