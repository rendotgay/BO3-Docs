# ClearPotentialThreat

Clear the potential threat direction. See SetPotentialThreat for more info on potential threats

Example:
```c++
self ClearPotentialThreat( 90 )
```

### Parameters
Parameter | Summary
| - | - |
*dir | direction of the threat. This is the angle from which the NPC should expect danger

<sub>* = mandatory</sub>
