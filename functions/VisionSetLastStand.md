# VisionSetLastStand

Sets players' laststand vision.  Optionally give a transition time from the current vision.

Example:
```c++
VisionSetLastStand( "armada", 1.5 )
```

### Parameters
Parameter | Summary
| - | - |
*vision name | Vision to transition to.
[transition time] | How long, in seconds, to take transitioning to the new vision set if a transition is possible.  Default is one second.

<sub>* = mandatory</sub>
