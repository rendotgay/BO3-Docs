# Target_SetShader

Changes the material of a target.  The target must have already been created with target_set()

Example:
```c++
Target_SetShader( enemy_chopper, "locked_on_shader" )
```


### Parameters
Parameter | Summary
| - | - |
*target entity | The entity that is the target
*materialname | The shader for the quad drawn over the target

<sub>* = mandatory</sub>
