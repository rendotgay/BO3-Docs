# GetUIModel

Returns a UIModel from the parent model passed in and the name of the model to retrieve.

Example:
```c++
GetUIModel( gameScoreModel, "enemeyScore" )
```

### Parameters
Parameter | Summary
| - | - |
*parentModel | The parent model of the model to retrieve.
*lookupModelName | The name of the model to retrieve.

<sub>* = mandatory</sub>
