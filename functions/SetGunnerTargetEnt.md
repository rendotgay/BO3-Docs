# SetGunnerTargetEnt

Set the target entity for this vehicle gunner.

Example:
```c++
eLiberator SetGunnerTargetEnt( eFlaktarget, (0,0,0), 2 )
```

### Parameters
Parameter | Summary
| - | - |
*target entity | The target.
[target offset] | The target offset.
[gunner index] | Which gun, 0-3.

<sub>* = mandatory</sub>
