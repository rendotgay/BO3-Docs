# Spawn

Spawn an entity.

Example:
```c++
Spawn( "trigger_radius", trigger_origin )
```


**Returns:** [`entity`](../types/entity.md)

### Parameters
Parameter | Summary
| - | - |
*classname | The classname of the entity to spawn
*origin | The position to spawn at
[spawnflags] | Optional spawn flags (up to 5)

<sub>* = mandatory</sub>
