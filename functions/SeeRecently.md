# SeeRecently

Check to see if the AI saw the given sentient in the past &lt;time&gt;. If called on a non-sentient, just does a cansee check

Example:
```c++
if( self SeeRecently( player ) )
```


### Parameters
Parameter | Summary
| - | - |
*target | (entity) The entity to check.
*time | (int) seconds.

<sub>* = mandatory</sub>
