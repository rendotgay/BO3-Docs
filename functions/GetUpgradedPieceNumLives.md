# GetUpgradedPieceNumLives

Returns the number of lives, or pull reps it should take a zombie to dislodge this piece, if it's upgraded.

Example:
```c++
piece.numLives = ent GetUpgradedPieceNumLives(1)
```

### Parameters
Parameter | Summary
| - | - |
*piece number | The piece index.

<sub>* = mandatory</sub>
