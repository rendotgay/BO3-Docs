# LinkToCamera

Links an entity directly to the camera.  Good for scripted player arms.

Example:
```c++
swimming_arms LinkToCamera()
```


### Parameters
Parameter | Summary
| - | - |
*linkType | , 0 - pitch only, 1 - yaw only, 2 - roll only, 3 - swimming, 4 - full, 5 - 3d compass.  Full is the default if not specified."
[offset] | Vector of the offset

<sub>* = mandatory</sub>
