# GetBuildKitWeapon

Get weapon object (including attachments) for the weapon from the player's build kit

Example:
```c++
weapon = self GetBuildKitWeapon( ar_standard )
```


### Parameters
Parameter | Summary
| - | - |
*weapon | weapon object
upgraded | bool whether this should be the upgraded version. If so extclip and fmj will be added

<sub>* = mandatory</sub>
