# IsTouchingVolume

Returns true if entity is touching the volume.

Example:
```c++
if ( crate_ent IsTouchingVolume( origin + (0,0,40), mins, maxs ) )
```

### Parameters
Parameter | Summary
| - | - |
*origin | Origin of the volume
*volume mins | The min extents of the volume
*volume maxs | The max extents of the volume

<sub>* = mandatory</sub>
