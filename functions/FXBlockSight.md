# FXBlockSight

Uses a general entity and a radius to set a line of sight blocker on the server for some AI. Destroyed when the entity is freed.

Example:
```c++
FXBlockSight( fxent, 64 )
```


### Parameters
Parameter | Summary
| - | - |
*entity | the entity to use for the origin for tje line of sight blocker
*radius | the radius of the 'smoke screen' blocker

<sub>* = mandatory</sub>
