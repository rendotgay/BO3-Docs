# SetLoopState

set pitch and volume per entity/alias

Example:
```c++
car SetLoopState( "peugeot_idle_low", 0.5, 1.0, 0.5, 0.2 )
```

### Parameters
Parameter | Summary
| - | - |
*aliasname | The sound alias to play
*attenuation | The desired volume attenuation
*pitch | The desired pitch
[attenuation_rate] | The desired attenuation rate
[pitch_rate] | The desired pitch rate

<sub>* = mandatory</sub>
