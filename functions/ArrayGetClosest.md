# ArrayGetClosest

Returns the entity/vector that is closest to the &lt;origin&gt; from the &lt;array&gt;.

Example:
```c++
ent = ArrayGetClosest( player.origin, a_enemies )
```


### Parameters
Parameter | Summary
| - | - |
*origin | The origin to get closest from
*array | The array to search in
[dist] | The minimum distance to check when finding closest (has to be positive)

<sub>* = mandatory</sub>
