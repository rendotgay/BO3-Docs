# SetGenericSceneValue

Sets one of the 16 generic scene vectors used by SW shaders.

Example:
```c++
SetGenericSceneValue(0, 20)
```

### Parameters
Parameter | Summary
| - | - |
*localClientNum | Local client number.
*index | The scene vector index ( 0 - 15 )
*value | Some generic float used by SW shaders.

<sub>* = mandatory</sub>
