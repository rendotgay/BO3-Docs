# SetYawSpeed

Sets the yaw speed for this vehicle.

Example:
```c++
self SetYawSpeed( 120, 60 )
```


### Parameters
Parameter | Summary
| - | - |
*speed | The speed of the vehicle in angles per second
*acceleration | The acceleration of the vehicle in angles per second^2
[deceleration] | Deceleration. If not specified, set to be equal to acceleration

<sub>* = mandatory</sub>
