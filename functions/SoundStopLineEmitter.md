# SoundStopLineEmitter

Stops the sound playing from start point to end point

Example:
```c++
SoundStopLineEmitter( "wpn_micro_turret_loop", start, end )
```

### Parameters
Parameter | Summary
| - | - |
*alias | The sound alias name
*origin 1 | start point
*origin 2 | end point

<sub>* = mandatory</sub>
