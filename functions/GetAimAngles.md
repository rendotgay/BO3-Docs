# GetAimAngles

Returns an array with the angles of the aimTag and the relative aiming delta angles from this orientation.

Example:
```c++
aim_angles = ent GetAimAngles( shootPos, "tag_aim", (0, 30, 0), (15, 10, 0), (60, 60, 0), (5, 5, 0), 11 )
```

| - | - |
*targetPos | 
*aimTag | 
*prevAimTagAngles | 
*prevAimAngles | 
*maxAngles | 
*maxAimDeltas | 
*maxTagDelta |
