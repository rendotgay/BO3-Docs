# SetTeamSpyplane

Sets whether a team has Spyplane or not

Example:
```c++
SetTeamSpyplane( "allies", true )
```

### Parameters
Parameter | Summary
| - | - |
*team | The name of a team. Must be either Must be 'axis', 'allies', 'team3' or 'none'.
*availability | Whether the team has Spyplane; defaults to false

<sub>* = mandatory</sub>
