# PopulateScriptDebugMenu

Sets a UIModel from the model being passed in.

Example:
```c++
PopulateScriptDebugMenu( localClientNum, menu_items_array )
```

### Parameters
Parameter | Summary
| - | - |
*localClientNum | Number of the local client on the machine.
*array | Array of menu item names for the debug menu.

<sub>* = mandatory</sub>
