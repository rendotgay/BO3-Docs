# ArraySortClosest

Returns a sorted array of entities by closest from &lt;origin&gt



### Parameters
Parameter | Summary
| - | - |
*array | array to sort
*origin | the origin from which to compare
[maxReturnCount] if set then only the first max count of entities will be returned | 
[minDist] | if set then only the entities further than <minDist> will be considered
[maxDist] | if set then only the entities closer then <maxDist will be considered

<sub>* = mandatory</sub>
