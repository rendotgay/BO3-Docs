# DirectionalHitIndicator

Create a directional hit indicator

Example:
```c++
attacker DirectionalHitIndicator( victimArray0, victimArray1 )
```


### Parameters
Parameter | Summary
| - | - |
*victims0 | A bit array of entities 0-31 that got hit
[victims1] | A bit array of entities 32-63 that got hit

<sub>* = mandatory</sub>
