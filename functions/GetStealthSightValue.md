# GetStealthSightValue

Get the current stealth sight value for this sentient seeing another sentient (floating point from 0.0 to 1.0)

Example:
```c++
aiGuy GetStealthSightValue( forOtherGuy )
```


### Parameters
Parameter | Summary
| - | - |
*otherSentient | other sentient you want to get the value on this sentient for

<sub>* = mandatory</sub>
