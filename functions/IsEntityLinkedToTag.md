# IsEntityLinkedToTag

Returns is the entity is linked to the parententity, ( on a specific tag optional )

Example:
```c++
if ( player IsEntityLinkedToTag( grenade, "j_head" ) )
```


### Parameters
Parameter | Summary
| - | - |
*entity | The entity that we are checking
[tag] | The tag we are checking

<sub>* = mandatory</sub>
