# SpawnPathNode

Spawn a dynamically created path node or cover node

**Returns:** [`pathnode`](../types/pathnode.md)

### Parameters
Parameter | Summary
| - | - |
*classname | The class name
*origin | The origin
*angles | The view angles
[key1] | Key 1
[value1] | Value for key 1
[key2] | Key 2
[value2] | Value for key 2

<sub>* = mandatory</sub>
