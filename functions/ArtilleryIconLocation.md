# ArtilleryIconLocation

Set the artillery icon location

Example:
```c++
ArtilleryIconLocation( self.origin, 1, true )
```

### Parameters
Parameter | Summary
| - | - |
*position | Position of icon
*teamNum | Team number
*isActive | Whether the icon is active or not
[isMortar] | Whether the artillery is a mortar
[clientNum] | The client number

<sub>* = mandatory</sub>
