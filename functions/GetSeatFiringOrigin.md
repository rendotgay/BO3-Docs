# GetSeatFiringOrigin

Retrieve the location at the end of the barrel where bullets, projectiles, etc. are emitted when the gun is fired.

Example:
```c++
turretBarrelOrigin = tank GetSeatFiringOrigin( 0 )
```


### Parameters
Parameter | Summary
| - | - |
*seat index | The seat to firing origin for

<sub>* = mandatory</sub>
