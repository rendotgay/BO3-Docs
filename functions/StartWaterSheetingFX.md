# StartWaterSheetingFX

Play a fullscreen water effect.

Example:
```c++
StartWaterSheetingFX(localClientNum, 8)
```

### Parameters
Parameter | Summary
| - | - |
*localClientNum | Local client on which to play the fx
[duration] | The number of seconds to play the fx before it stops automatically. Use 0 for infinite time. Defaults to 0.
[magnitude] | default is from a dvar (r_waterSheetingFX_magnitude)

<sub>* = mandatory</sub>
