# GetEquippedBodyForHero

Returns the name of the equipped body model of a specific hero

Example:
```c++
modelName = GetEquippedBodyForHero( localClientNum, SESSIONMODE_MULTIPLAYER, 0 )
```

### Parameters
Parameter | Summary
| - | - |
*localClientNum | The localClientNum of the player in question
*sessionMode | The character's sessionMode
*heroIndex | The index of the hero in the current game-mode's character table

<sub>* = mandatory</sub>
