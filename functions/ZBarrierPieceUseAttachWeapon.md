# ZBarrierPieceUseAttachWeapon

Sets the indexed piece to attach the zbarrier's weapon to tag_weapon (and tag_weapon_left if it's dual wield).

Example:
```c++
ent ZBarrierPieceUseAttachWeapon( 1 )
```

### Parameters
Parameter | Summary
| - | - |
*piece number | The piece index.

<sub>* = mandatory</sub>
