# WeaponFriendlyHacking

Returns the value of the friendlyHacking bit on the specified weapon of the specified player

Example:
```c++
currentWeapon = GetCurrentWeapon( localClientNum )
```


### Parameters
Parameter | Summary
| - | - |
*localClientNum | Local client number of the player
*localClientNum | Weapon to check

<sub>* = mandatory</sub>
