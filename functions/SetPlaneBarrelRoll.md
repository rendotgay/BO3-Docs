# SetPlaneBarrelRoll

Set the goal Roll direction for this plane. PLANE_ON_CURVE has priority over ROLL if this is not set.

Example:
```c++
level.plane SetPlaneBarrelRoll( 1, 5.0f )
```


### Parameters
Parameter | Summary
| - | - |
*number | The number of rolls
[time] | The goal roll time

<sub>* = mandatory</sub>
