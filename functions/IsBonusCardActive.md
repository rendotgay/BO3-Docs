# IsBonusCardActive

Returns if the bonus card is enabled and active in the class, cannot be called on a larry

Example:
```c++
primary = self IsBonusCardActive( BONUSCARD_PRIMARY_GUNFIGHTER_INDEX, self.class_num )
```


### Parameters
Parameter | Summary
| - | - |
*bonuscard | BonusCard index from bonuscard.gsh
*classnum | The current class num of the player

<sub>* = mandatory</sub>
