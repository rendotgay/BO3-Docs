# SetGunnerTargetVec

Set the target position for this vehicle turret gunner.

Example:
```c++
level.flak1 SetGunnerTargetVec( vec1, 1 )
```

### Parameters
Parameter | Summary
| - | - |
*target position | vecter to aim at/ the position of the target
[gunner index] | which gun 0-3

<sub>* = mandatory</sub>
