# GetTotalAmmo

Returns the total amount of ammo the client has for the currently equipped weapon

Example:
```c++
ammoCount = GetTotalAmmo( localClientNum, "ak47_mp" )
```


### Parameters
Parameter | Summary
| - | - |
*localClientNum | Client number of the player
*weaponName | Name of the desired weapon to check

<sub>* = mandatory</sub>
