# SetWeaponCosts

Sets the cost of a weapon for use by prompts

Example:
```c++
SetWeaponCosts( weapon, weapon.cost, weapon.ammoCost, weapon.secretshopperCost )
```


### Parameters
Parameter | Summary
| - | - |
*local client num | The local client num
*weapon | The weapon to add
*weaponCost | Cost to buy the weapon
*ammoCost | Cost to buy ammo
*playerAmmoCost | Cost to buy ammo through secret shopper

<sub>* = mandatory</sub>
