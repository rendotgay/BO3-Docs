# GetFXFromSurfaceTable

Returns the fx for a given surface type from the surfacefxtable.

Example:
```c++
fx = GetFXFromSurfaceTable( fx_surface_table, surface_type )
```

### Parameters
Parameter | Summary
| - | - |
*surfacefxtable | Surface fx table.
*surface type | Surface type

<sub>* = mandatory</sub>
