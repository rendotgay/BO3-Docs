# UseBuildKitWeaponModel

Use a build kit weapon model

Example:
```c++
wall_weapon_model UseBuildKitWeaponModel( player, magicbox.weapon )
```

### Parameters
Parameter | Summary
| - | - |
*player | The player whose build kit data will be used
*weapon | The weapon to base the build kit on
[camoIndex] | The index of the camo to override the build kit's camo with
upgraded | bool whether this should be the upgraded version. If so extclip and fmj will be added

<sub>* = mandatory</sub>
