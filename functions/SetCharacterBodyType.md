# SetCharacterBodyType

Sets the character body type for the player.  Resets all other customization to default for the new body type.

Example:
```c++
self SetCharacterBodyType( 1 )
```

### Parameters
Parameter | Summary
| - | - |
*character body type index | The index of the character body type
character head type index | The index of the character head type

<sub>* = mandatory</sub>
