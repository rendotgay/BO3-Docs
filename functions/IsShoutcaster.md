# IsShoutcaster

Returns 1 if the local client is shoutcasting

Example:
```c++
if ( IsSpectating( localClientNum, false ) )
```


### Parameters
Parameter | Summary
| - | - |
*localClientNum | Number of the local client to check
[checkFistPerson] | Check if we want first person spectating to be not considered as spectate

<sub>* = mandatory</sub>
