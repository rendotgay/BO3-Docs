# SetVehicleAvoidance

Turns on and off vehicle avoidance

Example:
```c++
vehicle SetVehicleAvoidance( true, 50 )
```


### Parameters
Parameter | Summary
| - | - |
*enabled | True to enable avoidance, false otherwise
[radius] | The avoidance radius
[priority] | Vehicles will avoid everything that is the same priority or higher.  Default priority is 1.

<sub>* = mandatory</sub>
