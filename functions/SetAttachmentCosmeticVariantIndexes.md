# SetAttachmentCosmeticVariantIndexes

Sets the previously calculated SetAttachmentCosmeticVariantIndexes of the entity. The entity should not be a player

Example:
```c++
killcamEnt SetAttachmentCosmeticVariantIndexes( acvi )
```

### Parameters
Parameter | Summary
| - | - |
*acvi | The AttachmentCosmeticVariantIndexes to set

<sub>* = mandatory</sub>
