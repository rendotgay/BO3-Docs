# GetDvarVector

Gets the value of a dvar.

Example:
```c++
GetDvarVector( "r_eyesAdjust", (1,0,0) )
```

### Parameters
Parameter | Summary
| - | - |
*dvar | The dvar name as a string or hash.
[default_value] | The value to be returned if the dvar doesn't exist, must be an vector.

<sub>* = mandatory</sub>
