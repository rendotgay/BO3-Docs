# EnableFrontendTokenLockedWeaponOverlay

Enable or disable the draw of the purchaseable gun postfx in script.

Example:
```c++
entity EnableFrontendTokenLockedWeaponOverlay( 0, 0 )
```


### Parameters
Parameter | Summary
| - | - |
*localClientNum | 
*enable | 0-disable 1-enable

<sub>* = mandatory</sub>
