# SetWeaponFriendlyTargetLocked

Set the weapon lockon completion percentage

Example:
```c++
self SetWeaponFriendlyTargetLocked( weapon, 0.0 )
```

### Parameters
Parameter | Summary
| - | - |
*weapon | The weapon (player should have this weapon)
*isFriendlyTargetLocked | Set to true if a friendly is target locked

<sub>* = mandatory</sub>
