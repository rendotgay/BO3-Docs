# Objective_SetStencil

Sets the objective's stencil for the icon.

Example:
```c++
Objective_SetStencil( localClientNum, clientObjID, true )
```


### Parameters
Parameter | Summary
| - | - |
*localClientNum | The local client that is using the objective
*objective_number | The ID of the objective to alter
*stencil | Set to true if you want the objective to stencil

<sub>* = mandatory</sub>
