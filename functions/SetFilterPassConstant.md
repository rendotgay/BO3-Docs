# SetFilterPassConstant

Sets the filter pass constant

Example:
```c++
SetFilterPassConstant( self.localClientNum, 1, 12, 12, 2.5 )
```


### Parameters
Parameter | Summary
| - | - |
*filterIndex | Must be between 0 - 3
*passIndex | Must be between 0 - 16
*constantIndex | Must be between 0 - 16
*value | The value for the constant

<sub>* = mandatory</sub>
