# GetDvarString

Gets the value of a dvar.

Example:
```c++
GetDvarString( "r_eyesAdjust", "yes" )
```

### Parameters
Parameter | Summary
| - | - |
*dvar | The dvar name as a string or hash.
[default_value] | The value to be returned if the dvar doesn't exist, must be a string.

<sub>* = mandatory</sub>
