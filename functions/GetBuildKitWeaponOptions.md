# GetBuildKitWeaponOptions

Get weapon options for the weapon from the player's build kit

Example:
```c++
weapon_options = self GetBuildKitWeaponOptions( ar_standard, packapunch_camo_index )
```


### Parameters
Parameter | Summary
| - | - |
*weapon | weapon object
[camo_index] | camo index to override the one from the build kit, typically a packapunch camo index

<sub>* = mandatory</sub>
