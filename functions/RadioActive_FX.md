# RadioActive_FX

Plays RadioActive/Fire FX

Example:
```c++
entity RadioActive_FX(0, true)
```


### Parameters
Parameter | Summary
| - | - |
*localClientNum | Client number of the player
true | /false. Default is true.
Blur | . 0 - 5. Default is 0.
Speed | . 0 - 2. Default is 1.
Intensity | . 0 - 5. Default is 2.

<sub>* = mandatory</sub>
