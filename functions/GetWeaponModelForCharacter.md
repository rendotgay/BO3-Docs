# GetWeaponModelForCharacter

Returns the name of the weapon model of at the given index

Example:
```c++
modelName = GetWeaponModelForCharacter( 0, "mp" )
```


### Parameters
Parameter | Summary
| - | - |
*characterIndex | The index of the character in the current game-mode's character table
*sessionmode | Session Mode Index

<sub>* = mandatory</sub>
