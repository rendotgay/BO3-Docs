# SetSAnim

Set the anim shot to play on a static model

Example:
```c++
thing SetSAnim( anim, shot, 1 )
```


### Parameters
Parameter | Summary
| - | - |
*string anim name | name of a SANIM asset
[string shot name] | shot name of the sub animation to play, pass in <int> 0 to play the first one.
[float time] | frame to start playing on, default is 0.

<sub>* = mandatory</sub>
