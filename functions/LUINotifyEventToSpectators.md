# LUINotifyEventToSpectators

Sends a notify from script to LUI.

Example:
```c++
LUINotifyEventToSpectators( &"update_objectives", 1, &"LEVEL_GOTO_NEXT_PLACE" )
```


### Parameters
Parameter | Summary
| - | - |
*lui event name | as an IString
*num of args | number of parameters
*args | argument to pass to LUI

<sub>* = mandatory</sub>
