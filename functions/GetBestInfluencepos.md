# GetBestInfluencepos

returns the influence map position for ai, this is custom logic to return the closest posistion between the passed in thresholds

Example:
```c++
dir = self GetBestInfluencePos( index, low, high)
```


### Parameters
Parameter | Summary
| - | - |
*index | which influence map to use
*low | the low threshold value to check against
*high | the high threshold value to check against

<sub>* = mandatory</sub>
