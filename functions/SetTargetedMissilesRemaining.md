# SetTargetedMissilesRemaining

Set targeted entity missiles remaining, for use on the client side in the UI.

Example:
```c++
self SetTargetedMissilesRemaining( weapon, 0.0 )
```


### Parameters
Parameter | Summary
| - | - |
*weapon | The weapon (player should have this weapon)
*missilesRemaining | The amount of missiles remaining to destroy the target

<sub>* = mandatory</sub>
