# MagicGrenadePlayer

Creates a "Magic" grenade from a player.

Example:
```c++
MagicGrenadePlayer( "sticky_grenade_mp", self.origin, toss_velocity )
```


**Returns:** [`entity`](../types/entity.md)

### Parameters
Parameter | Summary
| - | - |
*weapon | The grenade weapon to use
*origin | The starting point of the grenade toss
*velocity | The initial velocity vector for the grenade movement

<sub>* = mandatory</sub>
