# SetThreatBias

Set threat bias of second group against first group

Example:
```c++
SetThreatBias( "axis", "flanking_team", 120 )
```

### Parameters
Parameter | Summary
| - | - |
*against group | (string) threat bias group against.
*for group | (string) threat bias group for.
*threat bias | (int) threat bias value.

<sub>* = mandatory</sub>
