# SetTurretTargetRelativeAngles

Set the target entity for this vehicle turret.

Example:
```c++
eFlak88 SetTurretTargetEnt( eFlaktarget, eFlaktarget.origin )
```

### Parameters
Parameter | Summary
| - | - |
*relative angles | The target angle
[turret index] | 0 for turret, 1-4 for gunners

<sub>* = mandatory</sub>
