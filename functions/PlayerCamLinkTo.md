# PlayerCamLinkTo

Sets the player's vision to look through another entity.

Example:
```c++
level.player2 CameraLinkTo( cameraUpstairs, "tag_player" )
```


### Parameters
Parameter | Summary
| - | - |
*entity | Entity to look through.
*tagname | Name of tag on host model to attach to.

<sub>* = mandatory</sub>
