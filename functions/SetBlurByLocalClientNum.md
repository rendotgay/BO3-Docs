# SetBlurByLocalClientNum

Sets the blur for the local client given

Example:
```c++
SetBlurByLocalClientNum( localClientNum, curr_info.magnitude, curr_info.transition_in )
```

### Parameters
Parameter | Summary
| - | - |
*localClientNum | The local client number
*blurFinalValue | The blur value (must be positive)
*blurTime | The time to blur (must be positive)

<sub>* = mandatory</sub>
