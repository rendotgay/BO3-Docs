# ForceGameModeMappings

Forces the users controller mappings for a specific game mode (or resets to user defaults)

Example:
```c++
ForceGameModeMappings( localClientNum, "default" )
```

### Parameters
Parameter | Summary
| - | - |
*localClientNum | Client number to play the animation on
*modeName | Name of the mode to switch to or "default" for user defaults

<sub>* = mandatory</sub>
