# BO3 Syntax Highlight

Add GSC, CSC, and GSH syntax highlighting to Sublime Text

1. Navigate to `Call of Duty Black Ops III\docs_modtools`
2. In Sublime Text, select `Preferences > Browse Packages`
3. Navigate into the `Typescript` folder, creating it if missing
4. Extract the contents of `gsc_sublime.zip` from `Call of Duty Black Ops III\docs_modtools` into the `Typescript` folder
