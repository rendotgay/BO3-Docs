# PlayerPhysicsTraceIgnoreEnt

Does a physics trace and returns the hit point.

Example:
```c++
hitp = PlayerPhysicsTraceIgnoreEnt(player.origin + (0,0,72), player.origin + (0,0,-500))
```

### Parameters
Parameter | Summary
| - | - |
*start | starting position
*end | ending position

<sub>* = mandatory</sub>
