# SetCameraSpikeCamStatic

Sets whether a client sees static on the camera-spike cam

Example:
```c++
SetCameraSpikeCamStatic( localClientNum, 1 )
```


### Parameters
Parameter | Summary
| - | - |
*localClientNum | Client number of the camera-spike cam owner
*active | 0 if inactive, 1 if active

<sub>* = mandatory</sub>
