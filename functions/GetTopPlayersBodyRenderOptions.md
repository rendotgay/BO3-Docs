# GetTopPlayersBodyRenderOptions

Returns the render options for the body for the top player in the game by index

Example:
```c++
renderOptions = GetTopPlayersBodyRenderOptions( localClientNum, 2 )
```


### Parameters
Parameter | Summary
| - | - |
*localClientNum | The localClientNum of the player in question
*topPlayerIndex | The index of the top player

<sub>* = mandatory</sub>
