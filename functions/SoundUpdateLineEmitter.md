# SoundUpdateLineEmitter

Updates the sound playing from previous start point to previous end point

Example:
```c++
SoundUpdateLineEmitter( "wpn_micro_turret_loop", previousStart, previousEnd, start, end )
```


### Parameters
Parameter | Summary
| - | - |
*alias | The sound alias name
*previous origin 1 | start point
*previous origin 2 | end point
*origin 1 | start point
*origin 2 | end point

<sub>* = mandatory</sub>
