# DeleteFX

Delete a spawned effect.

Example:
```c++
deletefx (0, fxRef)
```


### Parameters
Parameter | Summary
| - | - |
*localClientNum | client on which to spawn effect.
*effect id | The effect id returned by spawnfx
[deleteImmediate] | whether effect will be deleted immediately. Default is 1.

<sub>* = mandatory</sub>
