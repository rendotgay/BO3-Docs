# GetSeatFiringAngles

Retrieve the angles specifying the direction in which bullets, projectiles, etc. will be emitted from the weapon when it is fired.

Example:
```c++
turretBarrelAngles = tank GetSeatFiringAngles( 0 )
```

### Parameters
Parameter | Summary
| - | - |
*seat index | The seat to firing angles for

<sub>* = mandatory</sub>
