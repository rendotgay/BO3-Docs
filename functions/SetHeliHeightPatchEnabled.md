# SetHeliHeightPatchEnabled

Enabled or disables the specified heli height lock path

Example:
```c++
SetHeliHeightPatchEnabled( "area51", 0 )
```


### Parameters
Parameter | Summary
| - | - |
*targetname | All heli height lock patches with this target name
*enabled | Turn it on or off
[player] | If defined, only modify the setting for that player

<sub>* = mandatory</sub>
