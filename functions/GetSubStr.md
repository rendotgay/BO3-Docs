# GetSubStr

Returns the substring of characters &gt;= &lt;start index&gt; and &lt; &lt;end index&gt;. &lt;end index&gt; is optional.

Example:
```c++
name = getSubStr( playerwarp_string, 8 )
```


### Parameters
Parameter | Summary
| - | - |
*string | The input string
*start index | The start index of the substring
[end index] | The end index of the substring

<sub>* = mandatory</sub>
