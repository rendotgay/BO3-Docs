# ScaleOverTime

Set a hud element to scale over time.

Example:
```c++
other.progressbar ScaleOverTime( level.planttime, level.barsize, 8 )
```

### Parameters
Parameter | Summary
| - | - |
*time | The time to scale the element in seconds
*width | The new width of the material.
*height | The new height of the material.

<sub>* = mandatory</sub>
