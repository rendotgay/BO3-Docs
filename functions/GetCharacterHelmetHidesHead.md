# GetCharacterHelmetHidesHead

Returns whether or not the characters head should be hidden based on the helmet

Example:
```c++
hideHead = GetCharacterHelmetHidesHead( 0, 0, 0 )
```

### Parameters
Parameter | Summary
| - | - |
*characterIndex | The index of the character in the current game-mode's character table
*helmetIndex | The index of the helmet belonging to that character
*sessionMode | The session mode ( mp cp zm )

<sub>* = mandatory</sub>
