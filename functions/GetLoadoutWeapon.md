# GetLoadoutWeapon

Get the full weapon name (including attachments) of the item in the custom class

Example:
```c++
primary = self GetLoadoutWeapon( 0, primary )
```

### Parameters
Parameter | Summary
| - | - |
*custom Class | Class Num
*loadoutSlot | Loadout Slot (primary or secondary)

<sub>* = mandatory</sub>
