# SetFXIgnorePause

Tells an effect whether or not to respect a paused world state.

Example:
```c++
SetFXIgnorePause( localClientNum, fxId, 1 )
```


### Parameters
Parameter | Summary
| - | - |
*localClientNum | The local client the effect is playing on.
*effect id | The effect id returned by playFx
*ignore pause | Whether or not to respect a paused world state.

<sub>* = mandatory</sub>
