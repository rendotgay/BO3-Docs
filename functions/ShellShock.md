# ShellShock

Start a shell shock sequence for the entity for given duration.

Example:
```c++
self shellShock( "frag_grenade_mp", 0.2 )
```


### Parameters
Parameter | Summary
| - | - |
*shellshockname | 
*duration | duration in seconds. The duration must be between 0 and 60 seconds.
[allowReduceShellShockPerk] | true if shell shock perk should be reduced

<sub>* = mandatory</sub>
