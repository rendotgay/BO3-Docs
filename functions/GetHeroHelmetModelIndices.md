# GetHeroHelmetModelIndices

Returns the given hero's helmet models for the current session mode

Example:
```c++
heroHelmetModelNames = GetHeroHelmetModelIndices( heroIndex, "mp" )
```

### Parameters
Parameter | Summary
| - | - |
*heroIndex | The index of the hero
[sessionmode] | The session mode (cp,mp,zm)

<sub>* = mandatory</sub>
