# Objective_SetIconSize

Sets the objective's scale for the icon.

Example:
```c++
objective_SetIconSize( localClientNum, clientObjID, 50 )
```

### Parameters
Parameter | Summary
| - | - |
*localClientNum | The local client that is using the objective
*objective_number | The ID of the objective to alter
*scale | The value to scale the objective icon

<sub>* = mandatory</sub>
