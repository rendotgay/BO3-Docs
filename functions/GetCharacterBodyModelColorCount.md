# GetCharacterBodyModelColorCount

Returns the body model count for a character for the current session mode

Example:
```c++
heroBodyModelCount = GetCharacterBodyModelCount( characterIndex, "mp" )
```


### Parameters
Parameter | Summary
| - | - |
*characterIndex | The index of the character in the current game-mode's character table
*bodyStyleIndex | The index of the body style in the character
*sessionmode | Session Mode Index

<sub>* = mandatory</sub>
