# IsPointInNavVolume

Checks if a given point is within the specified navigation volume.

Example:
```c++
result = IsPointInNavVolume( (10,20,30), "small volume" )
```


### Parameters
Parameter | Summary
| - | - |
*position | Position to evaluate.
*navVolumeName | Either "navvolume_small" or "navvolume_big".

<sub>* = mandatory</sub>
