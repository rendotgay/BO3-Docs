# GetPointInBounds

Returns a point within an entity's bounds.

Example:
```c++
targetForwardPnt = helo GetPointInBounds( 1.0, 0.0, 0.0 )
```

### Parameters
Parameter | Summary
| - | - |
*ratio_ForwardBack | 1.0 is full forward, -1.0 is full backwards.
*ratio_LeftRight | 1.0 is full left, -1.0 is full right.
*ratio_UpDown | 1.0 is full up, -1.0 is full down.

<sub>* = mandatory</sub>
