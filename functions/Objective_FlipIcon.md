# Objective_FlipIcon

Flips the icon of the objective

Example:
```c++
Objective_FlipIcon( self.friendlyObjID )
```


### Parameters
Parameter | Summary
| - | - |
*localClientNum | The local client number
*objectiveID | The index of objective
[shouldFlipMaterial] | 1 if the material should be flipped, 0 otherwise

<sub>* = mandatory</sub>
