# GetTouchingVolume

Get the entities that are inside the bounds given

Example:
```c++
entities = GetTouchingVolume( self.origin, mins, maxs )
```

**Returns:** [`entity`](../types/entity.md)

### Parameters
Parameter | Summary
| - | - |
*origin | Position of bounding box
*mins | The minimum extents of the bounding box
*maxs | The maximum extents of the bounding box

<sub>* = mandatory</sub>
