# SwitchMap_Preload

Start preloading a map/level.  A 'switchmap_preload_finished' notify is sent to the level when its finished.

Example:
```c++
SwitchMap_Preload( "cp_sh_cairo" )
```

### Parameters
Parameter | Summary
| - | - |
*map name | the name of the map to preload

<sub>* = mandatory</sub>
