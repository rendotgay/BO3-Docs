# Objective_VisibleTeams

Sets the team that the objective is visible too for. Allows having different objectives for each team

Example:
```c++
Objective_VisibleTeams( 0, 01101101 )
```

### Parameters
Parameter | Summary
| - | - |
*objective_number | The ID of the objective to alter
*team | The team that the objective is for. Valid entries are 'allies', 'axis', 'team3', or 'none'

<sub>* = mandatory</sub>
