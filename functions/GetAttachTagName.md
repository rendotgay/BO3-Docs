# GetAttachTagName

Returns the tagname of the attached model at the given attachment slot

Example:
```c++
tag_name = self GetAttachTagName( index )
```


### Parameters
Parameter | Summary
| - | - |
*modelindex | The index of the model attached to the entity, starting at 0, so for instance 3 will get the fourth model attached. (integer)

<sub>* = mandatory</sub>
