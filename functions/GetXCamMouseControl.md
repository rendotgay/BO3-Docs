# GetXCamMouseControl

Get the current "yaw" and "pitch" of the mouse xcam.  Returns an array with two keys, "yaw" and "pitch".

Example:
```c++
xcammouse = GetXCamMouseControl( localclientnum ) ...
```


### Parameters
Parameter | Summary
| - | - |
*localClientNum | Client number of the controller you want the mouse yaw/pitch of

<sub>* = mandatory</sub>
