# SetExcludeTeamForTrigger

Set the team that this trigger will not react to. Reacts to all other teams.

Example:
```c++
self SetExcludeTeamForTrigger( game["attackers"] )
```

### Parameters
Parameter | Summary
| - | - |
*team name | The name of the team that the trigger will not respond to. Must be either 'axis', 'allies', 'team3', 'team4', 'team5', 'team6' or 'none'

<sub>* = mandatory</sub>
