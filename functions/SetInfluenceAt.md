# SetInfluenceAt

Sets the influence value on the influence map, returns true/false if the value was set.

Example:
```c++
SetInfluenceAt(influenceMapIndex, origin, influence)
```

### Parameters
Parameter | Summary
| - | - |
*influenceMapIndex | Index of the influence map
*origin | The location to set the influence value
*influence | The influence value to set

<sub>* = mandatory</sub>
