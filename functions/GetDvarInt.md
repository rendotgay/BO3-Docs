# GetDvarInt

Gets the value of a dvar.

Example:
```c++
GetDvarInt( "r_eyesAdjust", 1 )
```


### Parameters
Parameter | Summary
| - | - |
*dvar | The dvar name as a string or hash.
[default_value] | The value to be returned if the dvar doesn't exist, must be an integer.

<sub>* = mandatory</sub>
