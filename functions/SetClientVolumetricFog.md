# SetClientVolumetricFog

Set the client volumetric fog properties

Example:
```c++

```

### Parameters
Parameter | Summary
| - | - |
*startDist | The start distance of the volume
*halfwayDist | The halfway distance of the volume
*halfwayHeight | The halfway height of the volume
*baseHeight | The base height of the volume
*red | The red value
*green | The green value
*blue | The blue value
*fogColorScale | The color scale of the fog
[sunFogColorR] | The sun fog red value
[sunFogColorG] | The sun fog green value
[sunFogColorB] | The sun fog blue value
[sunFogDirX] | The sun direction X value
[sunFogDirY] | The sun direction Y value
[sunFogDirZ] | The sun direction Z value
[sunFogStartAng] | Angle from the sun position where the sun fades in
[sunFogEndAng] | Angle from the sun position where the sun has faded out
[] | 
[fogMaxOpacity] | The maximum opacity of the fog as a value between 0 and 1

<sub>* = mandatory</sub>
