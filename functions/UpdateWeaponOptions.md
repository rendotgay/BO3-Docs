# UpdateWeaponOptions

Update weapon options for a weapon that the player has

Example:
```c++
player UpdateWeaponOptions( "m1garand", player calcWeaponOptions() )
```

### Parameters
Parameter | Summary
| - | - |
*weapon name | The weapon name to give to the player
[weapon options] | options value computed using calcweaponoptions call

<sub>* = mandatory</sub>
