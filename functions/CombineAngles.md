# CombineAngles

Returns angles b reoriented by angles a.

Example:
```c++
crateAnglesInWorld = CombineAngles( boatAngles, crateAnglesInBoat )
```


### Parameters
Parameter | Summary
| - | - |
*angles a | Angles representing the orientation in which to place <angles b>
*angles b | Angles which will be reoriented

<sub>* = mandatory</sub>
