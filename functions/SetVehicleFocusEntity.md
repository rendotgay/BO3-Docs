# SetVehicleFocusEntity

Sets the focus entity for a player driven vehicle (currently only supported by helicopters).  This will attempt to make the vehicle orient towards the focus target.

Example:
```c++
player SetVehicleFocusEntity( focusEntity )
```

### Parameters
Parameter | Summary
| - | - |
*focus entity | The entity to focus on

<sub>* = mandatory</sub>
