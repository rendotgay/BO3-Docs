# GetFirstHeroOfGender

Returns the first hero of gender ("male" or "female")

Example:
```c++
characterIndex = GetFirstHeroOfGender( "male", "mp" )
```


### Parameters
Parameter | Summary
| - | - |
*gender | "male" or "female"
[sessionmode] | The session mode (cp,mp,zm)

<sub>* = mandatory</sub>
