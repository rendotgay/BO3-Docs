# PlayClientAirstrike

Plays the client sided part of the airstrike/bombrun

Example:
```c++
PlayClientAirstrike( pos, yaw, "marines", "axis", owner, "left", height )
```


### Parameters
Parameter | Summary
| - | - |
*position | The position of airstrike
*yaw | The yaw
*teamFaction | The team faction
*team | The team
*owner | The owner of airstike
*exittype | The exit type
*height | The height

<sub>* = mandatory</sub>
