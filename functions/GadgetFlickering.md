# GadgetFlickering

Tests a gadget is flickering and can optionally set flickering.

Example:
```c++
self GadgetFlickering( slot, true, 200 )
```

### Parameters
Parameter | Summary
| - | - |
*slot | The gadget slot to test or apply flickering state. (integer)
[flicker] | Set gadget to flickering state if true (boolean)
[length] | Set gadget flickering length (integer)

<sub>* = mandatory</sub>
