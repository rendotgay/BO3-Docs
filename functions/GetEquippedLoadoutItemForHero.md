# GetEquippedLoadoutItemForHero

Returns 1 if the hero's ability is equipped, 0 if the weapon is equipped.

Example:
```c++
isAbilityEquipped = GetEquippedLoadoutItemForHero( localClientNum, 0 )
```


### Parameters
Parameter | Summary
| - | - |
*localClientNum | The localClientNum of the player in question
*heroIndex | The index of the hero in the current game-mode's character table

<sub>* = mandatory</sub>
