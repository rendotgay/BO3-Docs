# GetDvarFloat

Gets the value of a dvar.

Example:
```c++
GetDvarFloat( "r_eyesAdjust", 1.0 )
```


### Parameters
Parameter | Summary
| - | - |
*dvar | The dvar name as a string or hash.
[default_value] | The value to be returned if the dvar doesn't exist, must be an float.

<sub>* = mandatory</sub>
