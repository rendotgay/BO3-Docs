# SetWeaponFriendlyTargeting

Set the weapon lockon completion percentage

Example:
```c++
self SetWeaponFriendlyTargeting( weapon, 0.0 )
```

### Parameters
Parameter | Summary
| - | - |
*weapon | The weapon (player should have this weapon)
*isFriendlyTargeting | Set to true if a friendly is targetting

<sub>* = mandatory</sub>
