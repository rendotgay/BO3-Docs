# SetWaveWaterHeight

Sets the height of one or all waters in a level

Example:
```c++
SetWaveWaterHeight( "huge_lake", 15.0 )
```


### Parameters
Parameter | Summary
| - | - |
*water name | The name of the water affected, or empty string for all
*height | The new height value in world coordinates

<sub>* = mandatory</sub>
