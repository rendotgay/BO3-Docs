# GetVRControllerPosition

gets the world-space position of the tracked VR controller

Example:
```c++
pos = self GetVRControllerPosition( 0 ) ...
```


### Parameters
Parameter | Summary
| - | - |
*localClientNum | Client number of the controller you want the position of
*handIdx | 0 for dominant hand, 1 for secondary hand. Secondary hand is not guaranteed to exist

<sub>* = mandatory</sub>
