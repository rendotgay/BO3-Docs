# NewDamageIndicatorHudElem

Create a new hud element for a particular client

Example:
```c++
self.kc_topbar = NewClientHudElem(self)
```

**Returns:** [`hudelem`](../types/hudelem.md)

### Parameters
Parameter | Summary
| - | - |
*client | The client for whom the hud element is created.

<sub>* = mandatory</sub>
