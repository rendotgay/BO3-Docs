# Obituary

Create an obituary for a character

Example:
```c++
Obituary( self, attacker, sWeapon, sMeansOfDeath )
```


### Parameters
Parameter | Summary
| - | - |
*victim | The victim entity
*attacker | The attacker entity
*weapon | The weapon name
*weapon | The means of death as a string

<sub>* = mandatory</sub>
