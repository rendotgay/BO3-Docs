# PlayTagFXSet

Play a tagfx set on ajn entity. Do not specify loop fx

Example:
```c++
PlayTagFXSet ( 0, self, smoke )
```


### Parameters
Parameter | Summary
| - | - |
*localClientNum | Local client on which to play the fx
*tagfxset | The string name of the tag fx set
*entity | The entity to play the effect on

<sub>* = mandatory</sub>
