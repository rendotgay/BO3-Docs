# GetAIFxName

Returns a FX name corresponding to the AI's aiFxBundle key/value pair.

Example:
```c++
fxName = self GetAIFxName( localClientNum, surfaceTable )
```

### Parameters
Parameter | Summary
| - | - |
*localClientNum | Local client to check
*aiFxBundleKey | Script bundle key defined within the AI's aiFxTable.

<sub>* = mandatory</sub>
