# LinkNodes

Connects node1 to node2. Call it again with arguments flipped if you want a two-way connection.

Example:
```c++
LinkNodes( node_moving_elev_left, node_moving_elev_right )
```


### Parameters
Parameter | Summary
| - | - |
*node1 | first node to link
*node2 | second node to link

<sub>* = mandatory</sub>
