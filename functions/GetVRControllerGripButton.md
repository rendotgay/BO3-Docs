# GetVRControllerGripButton

returns true if the grip button is pressed

Example:
```c++
holding = self GetVRControllerGripButton( 0 ) ...
```

### Parameters
Parameter | Summary
| - | - |
*localClientNum | Client number of the controller you want
*handIdx | 0 for dominant hand, 1 for secondary hand. Secondary hand is not guaranteed to exist

<sub>* = mandatory</sub>
