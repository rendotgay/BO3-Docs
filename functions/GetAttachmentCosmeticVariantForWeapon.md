# GetAttachmentCosmeticVariantForWeapon

Get the AttachmentCosmeticVariantIndexes data to be passed in to GetWeapon

Example:
```c++
acvi = self GetAttachmentCosmeticVariantForWeapon( class_num, "primary" )
```

### Parameters
Parameter | Summary
| - | - |
*custom Class | Class Num
*loadoutSlot | Loadout Slot (primary or secondary)

<sub>* = mandatory</sub>
