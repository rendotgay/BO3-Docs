# IsMapSubLevel

Returns whether or not the map is a sublevel of a mission

Example:
```c++
level.isSubLevel = IsMapSubLevel( "cp_mi_cairo_ramses2" )
```

### Parameters
Parameter | Summary
| - | - |
[currentmap] | the name of the current map.  If ommitted, use Dvar sv_mapname

<sub>* = mandatory</sub>
