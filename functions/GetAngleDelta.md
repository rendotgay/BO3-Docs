# GetAngleDelta

Get the angle delta of the animation

Example:
```c++
angleDelta = getAngleDelta( arrivalAnim, 0, normalizedLength )
```


### Parameters
Parameter | Summary
| - | - |
*animation name | Name of the animation
[start time] | Start time of the animation
[end time] | End time of the animation

<sub>* = mandatory</sub>
