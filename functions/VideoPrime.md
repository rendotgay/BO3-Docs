# VideoPrime

Preload the first second of a movie.  Call this several seconds before the movie needs to play to prevent any delay

Example:
```c++
VideoPrime( "MyAwesomeMovie" )
```


### Parameters
Parameter | Summary
| - | - |
*name | name of movie to play.  do not add a file suffix (like .mp4)
loop | Set this to true if the movie should automatically restart when it reaches the end.

<sub>* = mandatory</sub>
