# MagicGrenadeManualPlayer

creates a "Magic" grenade from a player

Example:
```c++
self MagicGrenadeManualPlayer( self.origin, target.origin, 2.0 )
```


**Returns:** [`entity`](../types/entity.md)

### Parameters
Parameter | Summary
| - | - |
*origin | The starting point of the grenade toss
*velocity | The initial velocity vector for the grenade movement
*weapon | The grenade weapon to base on
[time to blow] | The grenade fuse time in seconds

<sub>* = mandatory</sub>
