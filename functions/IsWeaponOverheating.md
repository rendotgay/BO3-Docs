# IsWeaponOverheating

Is the players Weapon Overheating?

Example:
```c++
self IsWeaponOverheating()
```


### Parameters
Parameter | Summary
| - | - |
[get heatval] | If positive pass back the heatval of the weapon (0 to 100).
[weapon] | If present check this specific weapon otherwise check the player's current weapon.

<sub>* = mandatory</sub>
