# SightTracePassed

Does a sight trace from start to end. Returns true if doesnt hit anything

Example:
```c++
passed = SightTracePassed( player.origin + ( 0, 0, 64 ), self.origin + ( 0, 0, 64 ), false, undefined )
```


### Parameters
Parameter | Summary
| - | - |
*start | The bullet start point
*end | The bullet end point
*hit characters | An entity to ignore
*ignore entity | An entity to ignore

<sub>* = mandatory</sub>
