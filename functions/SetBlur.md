# SetBlur

Blur the screen over a period of time.

Example:
```c++
setblur( 10.3, 3.0 )
```

### Parameters
Parameter | Summary
| - | - |
*target_blur | The final blur. The value is pixels for Gaussian blur at 640x480. Must be a floating point value greater than 0.
*time | Time in seconds

<sub>* = mandatory</sub>
