# SetShader

Set the material for this Hud Element

Example:
```c++
waypoint SetShader( "playbook_objective_stop", 15, 15 )
```


### Parameters
Parameter | Summary
| - | - |
*materialname | A the name of the material to set
[width] | The width of the material.
[height] | The height of the material.

<sub>* = mandatory</sub>
