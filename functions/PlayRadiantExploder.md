# PlayRadiantExploder

Play the specified Radiant exploder

Example:
```c++
PlayRadiantExploder( 0, "lights_on_exploder" )
```


### Parameters
Parameter | Summary
| - | - |
*localClientNum | The local client to play the effect on
*exploderId | The id of the exploder to be triggered.

<sub>* = mandatory</sub>
