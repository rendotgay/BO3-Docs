# FakeFire

Simulates gun fire.  Sound and radar blip.

Example:
```c++
decoy FakeFire( player, position, "turret_mp", 20 )
```


### Parameters
Parameter | Summary
| - | - |
*owner | The player who fakes fire
*origin | the origin of the fake fire
*weapon | The weapon used
*shot count | Number of shots fired

<sub>* = mandatory</sub>
