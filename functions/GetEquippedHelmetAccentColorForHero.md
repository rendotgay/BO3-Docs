# GetEquippedHelmetAccentColorForHero

Returns the name of the equipped helmet accents colors of a specific hero

Example:
```c++
colorArray = GetEquippedHelmetAccentColorForHero( localClientNum, 0 )
```

### Parameters
Parameter | Summary
| - | - |
*localClientNum | The localClientNum of the player in question
*sessionMode | The character session mode
*heroIndex | The index of the hero in the current game-mode's character table
*helmetIndex | The index of the helmet
*colorIndex | The index of the accent color
[getForLobbyClient] | If this is set then we are supposed to retrieve for lobby client

<sub>* = mandatory</sub>
