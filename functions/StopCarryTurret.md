# StopCarryTurret

UnLinks the auto turret from the player and sets it on the ground.

Example:
```c++
player StopCarryTurret( turret, origin, angles )
```

### Parameters
Parameter | Summary
| - | - |
*turret | turret entity
[origin] | The new origin of turret
[angles] | The new angles of turret

<sub>* = mandatory</sub>
