# SetCOD7DecodeFX

Set the hudelem string to display with a decode effect.

Example:
```c++
elem SetCOD7DecodeFX( 2000, 600 )
```

### Parameters
Parameter | Summary
| - | - |
*decodeTime | Time for each letter to decode.
*decayStart | Time to wait before starting the text decay effect.
*decayDuration | How long to spend decaying.

<sub>* = mandatory</sub>
