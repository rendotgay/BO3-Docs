# SetDynEntBodyRenderOptionsPacked

Sets the packed value of body render options (body type, style, colors)

Example:
```c++
SetDynEntBodyRenderOptionsPacked( gibEntity, 0 )
```

### Parameters
Parameter | Summary
| - | - |
*dynent | 
*packed render options | Usually returned value from GetBodyRenderOptionsPacked()

<sub>* = mandatory</sub>
